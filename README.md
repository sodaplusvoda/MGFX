# mgfx

Минималистичный графический движок на Си, оформленный как C Extension для Python. Позволяет создавать независимые окна через GLFW и управлять ими по строковым ID.

Вся логика работы с контекстом и реестром окон написана на Си мной, интерфейс вызова перенесен в Python C API (с этим помогла Gemini).

## Как протестировать
Пример работы и весь доступный функционал находятся в файле `test.py`.

В будущем возможно допилю.
